# 💬 DoChat

A modern desktop chat application built using **HTML, CSS, and JavaScript**.

DoChat is a personal project focused on creating a clean and modern communication platform with features inspired by popular messaging apps while developing its own unique identity.

---

## ✨ Features

### ✅ Chat
- Real-time chat interface (UI)
- Send messages
- Edit messages
- Delete messages
- Copy messages
- Reply to messages
- Pin / Unpin messages
- Search messages
- Emoji picker
- Image sharing (UI)
- User status
- Profile popup

### 🌐 Spaces
- Dedicated Spaces page
- Workspace layout
- Modern desktop design

### 📷 Moments
- Moments page
- Share Moment UI
- Moment cards
- Story viewer (In Progress)

### 🤖 AI
- AI page (UI)

### 🎮 Games
- Games page (UI)

### ⚙️ Settings
- Theme switching
- Profile settings
- User customization

---

## 🛠 Technologies

- JavaScript (Vanilla)
- LocalStorage
