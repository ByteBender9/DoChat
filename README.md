# 💬 DoChat

A modern desktop communication platform built with **HTML, CSS, and Vanilla JavaScript**.

DoChat is a personal project focused on creating a clean, desktop-inspired messaging experience while improving frontend development skills through real-world application design.

---

## ✨ Features

### 🏠 Home
- Modern dashboard
- Quick access cards
- Clean desktop interface

### 💬 Chat
- Modern messaging interface
- Multiple conversations
- Send messages
- Edit messages
- Delete messages
- Reply to messages
- Pin / Unpin messages
- Copy messages
- Message search
- Emoji picker
- Image sharing
- User profile popup
- Theme support

### 🌐 Spaces
- Dedicated Spaces workspace
- Favorite spaces
- Multiple spaces
- Dynamic workspace layout
- Create Space modal
- Workspace announcements
- Channels section
- Members section
- Shared files section

### 📷 Moments
- Story-style interface
- Recent moments list
- Story viewer
- Dynamic moment switching
- Reply input

### 🤖 AI
- AI Assistant interface
- Modern dashboard layout

### 🎮 Games
- Games hub
- Game library interface

### ⚙️ Settings
- User profile settings
- Theme switching
- Customization panel

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- LocalStorage
- Font Awesome
